特別ツアー

  • 光明洞窟 4時間ツアー
    コース
    仁川国際空港 → 光明洞窟 → 仁川国際空港
    運営時間
    第1＆2ターミナル 09:00~13:00

    月曜日には仁川市ツアーコースに置き換えられます。 （仁川国際空港 - 月尾島公園 - 新浦市場 - 仁川国際空港）

    光明洞窟（旧始興鉱山）は、日本占領から解放後、韓国の事業開発の進化の過程を保存しています。記録によると、1955年と1972年の間に、金52kgが採掘され、1972年には廃鉱されましたが、まだかなりの金鉱が洞窟の中に残っているものと推測されます。

寺院

  • 伝燈寺（チョンドゥンサ）、徳津鎮（トッチンジン） 4時間ツアー
    コース
    仁川国際空港 → 伝燈寺 → 徳津鎮 → 仁川空港
    運営時間
    第1＆2ターミナル 8:00~12:00

    伝燈寺は韓国で最も古い寺院の一つで、韓国に初めて仏教が伝えられた4世紀に建てられ、また13世紀、高麗の王の元宗の時代に建て直されました。仏教の復興に大きな役割を果たした評判の高い寺院として、国宝第178号の大雄殿と共に貴重な遺跡が残されています。

    徳津鎮は高麗時代（918-1392年）に、江華海峡を守るのに最も重要な場所であり、強い砲兵基地でした。そして 「丙寅(へいいん)洋擾」（1866年フランスの侵略）と 「辛未(しんみ)洋擾」（アメリカの侵略） 戦闘が行われた場所です。徳津鎮の銃眼胸壁と衛所は丙寅洋擾の時に破壊され、1970年に再建されました。

  • 清渓川（チョンゲチョン）、曹渓寺（チョゲサ） 4時間ツアー
    コース
    仁川国際空港 → 清渓川 → 曹渓寺 → 仁川国際空港
    運営時間
    第1＆2ターミナル 14:00~18:00

    ソウルの鍾路区と中区の間を流れる清渓川はソウルの600年の歴史と、韓国人の生活と記憶が添えています。昔の清渓川は陸橋に隠れていた小さな水路でしたが、都市開発プロジェクトを通じて、ソウルで最も愛される観光スポットになりました。

    曹渓寺は大韓仏教曹渓宗の主な寺院であり、現代の仏教運動の中心地です。巨大な大雄殿は、美しい装飾と黄金仏教像を誇っており、大雄殿の前に平和に立っている500年前の白松は天然記念物第9号として曹渓寺に自然の美しさを増しています。毎年の灌仏会には提灯祭りを開催しています。
