娱乐

  • 天堂之城艺术特伦斯 2小时 旅行
    流程
    仁川国际机场 → 天堂之城艺术特伦斯 → 仁川国际机场
    运行时间
    第1航班口 7:00~9:00 / 13:00~15:00
    第2航班口 7:00~9:00 / 13:00~15:00

    天堂之城是东北亚地区第一家追随韩流的度假胜地，全年为娱乐场所，俱乐部，美术馆，售货亭等娱乐场所提供便利的住宿和设施。天堂城美术馆共收藏有2700件世界闻名的草间弥生，德昂斯等艺术家的作品，通过巡展可以看到八个天堂城美术馆的代表作品。

  • BMW 驾车中心 2小时 旅行
    流程
    仁川国际机场 → BMW 驾车中心 → 仁川国际机场
    运行时间
    第1航班口 9:00~11:00/ 11:00~13:00
    第2航班口 15:00-17:00

    在韩国的宝马驾驶中心是亚洲第一个宝马驾驶中心，拥有33个足球场。该中心有几个赛车场，展览厅，宴会厅，初级校园和娱乐场所。与此同时，品牌体验中心展出了宝马，迷你，摩托车和宝马的老爷车，赛道由6门课程组成。

食物&生活风格

  • 松岛 (G-塔 + NC 立方体运河路) 2小时 旅行
    流程
    仁川国际机场 → 松岛 G-塔 →
    松岛 NC 立方体运河路 → 仁川国际机场
    运行时间
    第1航班口 14:00~16:00
    第2航班口 16:00~18:00

    G-塔位于松岛国际城的中央公园，是仁川的景点之一。 33层的观景台展现了360度松岛国际城的美景。空中花园位于29楼，松岛中央公园，仁川大桥，仁川西海滩180度观景。

    松岛NC立方运河道路于2013年8月开业，拥有独具特色的咖啡馆，品牌商店和文化活动，是松岛的瑰宝。百货店分为四个部分来代表四季，人造水道像运河一样建造，四个部分有浪漫的气氛。

特别旅行

  • 光明洞窟 4小时 旅行
    流程
    仁川国际机场 → 光明洞窟 → 仁川国际机场
    运行时间
    第1 & 2 航班口 09:00~13:00

    星期一光明洞窟不运行时，将换成仁川市旅行 (仁川国际机场-月尾岛公园-新浦市场 –仁川国际机场)

    光明洞窟 (前 始兴銧山) 在1972年关闭和从日本占领解放之后，我们正在维护韩国企业发展的演变。据记载，1955年至1972年期间共开采52公斤黄金，1972年关闭，但据推测仍有相当数量的金矿遗留在洞内。

寺庙

  • 传灯寺, 德津镇 4小时 旅行
    流程
    仁川国际机场 → 传灯寺 → 德津镇 → 仁川国际机场
    运行时间
    第1 & 2 航班口 8:00~12:00

    传灯寺被认为是韩国最古老的寺庙之一，始建于公元四世纪，当时佛教首次引进韩国。 13世纪韩国国王元宗在位期间重建，在佛教复兴中发挥了重要作用。第178号国宝，美丽的大元殿，高丽，朝鲜时代的珍贵遗物得以保存。

    德津镇在高丽时代守护江华海峡的重要地方（918-1392）。它是江华海峡最强的炮兵基地，是1866年入侵法国（1866年入侵法国）和1871年入侵美国的辛米洋扰战场的所在地。 德津镇的枪城和大门在辛未洋扰期间被毁坏并且在1970年重建。

  • 清溪川 曹溪寺 4小时 旅行
    流程
    仁川国际机场 → 清溪川 → 曹溪寺 → 仁川国际机场
    运行时间
    第1 & 2 航班口 14:00~18:00

    在钟路和首尔中区之间流动的清溪川，有着600年的首尔历史和在这条河流中生活的韩国人的生活和回忆。过去，清溪川是一条覆盖着立交桥的小水道，但通过城市发展项目，成为首尔最受欢迎和最受欢迎的景点之一。

    曹溪寺是朝鲜寮国主要的佛教寺庙，是现代佛教运动的中心。巨大的大雄殿拥有美丽的装饰品和金色的佛像，在大雄殿前安静地站立着的500年历史的白松树是9号天然纪念物，为自然之美增添了自然之美。每年寺庙都会举行一个元宵节，以纪念佛祖的生日。

购物

  • 明洞 南大门市场 5小时 旅行
    流程
    仁川国际机场 → 明洞→ 南大门市场 → 仁川国际机场
    运行时间
    第1 & 2 航班口 13:00~18:00

    首尔的购物中心，商店，免税店等都是明洞。这是来自世界各地的旅客可以享受韩国最新时尚和各种食物的地方。

    南大门市场是韩国最大的传统市场，除了服装，碗，厨房用具，玩具，登山装备，帽子，鲜花和人参以外，还有许多卖小商品的小店。
